Subsequence Generator (Bitmasking)

The Concept: To find all possible subsequences of a string of length $n$, we can map every character to a specific bit in a binary number. By counting from $0$ to $(2^n - 1)$, the binary representation of our current count gives us every possible combination of "Take" (1) and "Skip" (0) decisions.

The Code Implementation

This method completely avoids recursion. It uses the left shift operator (1 << n) to find the upper bounds, and the Bitwise AND operator (&) to check if the $i$-th bit is turned on.

public static void generateSubsequences(String s) {
    int n = s.length();
    int totalMasks = 1 << n; // Calculates 2^n
    
    // Count from 0 up to (2^n - 1)
    for (int mask = 0; mask < totalMasks; mask++) {
        StringBuilder sub = new StringBuilder();
        
        // Check each bit position
        for (int i = 0; i < n; i++) {
            // If the i-th bit in 'mask' is a 1, take the character
            if ((mask & (1 << i)) != 0) {
                sub.append(s.charAt(i));
            }
        }
        System.out.println("Subsequence: {" + sub.toString() + "}");
    }
}

    int n = strlen(s);
    int totalMasks = 1 << n; // Calculates 2^n
    
    // Count from 0 up to (2^n - 1)
    for (int mask = 0; mask < totalMasks; mask++) {
        printf("Subsequence: {");
        // Check each bit position
        for (int i = 0; i < n; i++) {
            // If the i-th bit in 'mask' is a 1, take the character
            if ((mask & (1 << i)) != 0) {
                printf("%c", s[i]);
            }
        }
        printf("}\n");
    }
}
def generate_subsequences(s):
    n = len(s)
    total_masks = 1 << n # Calculates 2^n
    
    # Count from 0 up to (2^n - 1)
    for mask in range(total_masks):
        sub = []
        # Check each bit position
        for i in range(n):
            # If the i-th bit in 'mask' is a 1, take the character
            if (mask & (1 << i)) != 0:
                sub.append(s[i])
                
        print(f"Subsequence: {{ {''.join(sub)} }}")
