Problem 2: Longest Common Subsequence (LCS)

The Setup: You are given two strings, String X of length m and String Y of length n. Your task is to find the length of the longest subsequence that is present in both strings.

Algorithmic Paradigms

1. The Brute-Force Approach

Brute-force relies on exhaustive search. It recursively checks characters from the end. If they don't match, it spawns two new recursive branches. Generating all subsequences of one string and verifying them in the other leads to a massive O(m · 2n) or O(n · 2m) decision tree.

int lcs_recursive(char* X, char* Y, int m, int n) {
    if (m == 0 || n == 0) return 0;
    
    // Match found
    if (X[m-1] == Y[n-1]) {
        return 1 + lcs_recursive(X, Y, m-1, n-1);
    }
    // Mismatch: Branch into two recursive paths
    else {
        int branch1 = lcs_recursive(X, Y, m, n-1);
        int branch2 = lcs_recursive(X, Y, m-1, n);
        return (branch1 > branch2) ? branch1 : branch2;
    }
}
def lcs_recursive(X, Y, m, n):
    if m == 0 or n == 0:
        return 0
        
    # Match found
    if X[m-1] == Y[n-1]:
        return 1 + lcs_recursive(X, Y, m-1, n-1)
    # Mismatch: Branch into two recursive paths
    else:
        branch1 = lcs_recursive(X, Y, m, n-1)
        branch2 = lcs_recursive(X, Y, m-1, n)
        return max(branch1, branch2)
public static int lcsRecursive(String X, String Y, int m, int n) {
    if (m == 0 || n == 0) return 0;
    
    // Match found
    if (X.charAt(m-1) == Y.charAt(n-1)) {
        return 1 + lcsRecursive(X, Y, m-1, n-1);
    } 
    // Mismatch: Branch into two recursive paths
    else {
        int branch1 = lcsRecursive(X, Y, m, n-1);
        int branch2 = lcsRecursive(X, Y, m-1, n);
        return Math.max(branch1, branch2);
    }
}

2. The Greedy Approach

Why Greedy Fails Here: The Greedy approach lacks foresight. If you greedily match the first common character and move forward, you might lock yourself out of a much longer subsequence later.

// Naive Greedy LCS (FAILS for optimal answers)
int lcs_greedy_fails(char* X, char* Y, int m, int n) {
    int count = 0;
    int j_start = 0;
    
    for (int i = 0; i < m; i++) {
        for (int j = j_start; j < n; j++) {
            if (X[i] == Y[j]) {
                count++;
                j_start = j + 1; // Eagerly lock in the match
                break;
            }
        }
    }
    return count; 
}

3. The Dynamic Programming (DP) Approach

Dynamic Programming solves overlapping subproblems and stores the results in a 2D table so no character comparison is ever calculated twice. Time and space drop to O(m × n).

int lcs_dp(char* X, char* Y, int m, int n) {
    int L[m+1][n+1];
    
    for (int i = 0; i <= m; i++) {
        for (int j = 0; j <= n; j++) {
            if (i == 0 || j == 0)
                L[i][j] = 0;
            else if (X[i-1] == Y[j-1])
                L[i][j] = L[i-1][j-1] + 1; // Match: Add diagonal
            else
                L[i][j] = (L[i-1][j] > L[i][j-1]) ? L[i-1][j] : L[i][j-1]; // Max of left/above
        }
    }
    return L[m][n];
}
def lcs_dp(X, Y):
    m, n = len(X), len(Y)
    L = [[0] * (n + 1) for _ in range(m + 1)]
    
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if X[i-1] == Y[j-1]:
                L[i][j] = L[i-1][j-1] + 1 # Match: Add diagonal
            else:
                L[i][j] = max(L[i-1][j], L[i][j-1]) # Max of left/above
                
    return L[m][n]
public static int lcsDP(String X, String Y) {
    int m = X.length(), n = Y.length();
    int[][] L = new int[m + 1][n + 1];
    
    for (int i = 0; i <= m; i++) {
        for (int j = 0; j <= n; j++) {
            if (i == 0 || j == 0) {
                L[i][j] = 0;
            } else if (X.charAt(i-1) == Y.charAt(j-1)) {
                L[i][j] = L[i-1][j-1] + 1; // Match: Add diagonal
            } else {
                L[i][j] = Math.max(L[i-1][j], L[i][j-1]); // Max of left/above
            }
        }
    }
    return L[m][n];
}
