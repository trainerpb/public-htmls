Greedy Coin Change Simulator

The Coin Exchange Problem aims to find the minimum number of coins needed to make a specific target amount. It can be approached using Greedy Algorithms (which are extremely fast but can fail on arbitrary, non-standard coin systems) or Dynamic Programming (DP) (which guarantees an optimal solution by checking overlapping subproblems).

Note: The Dynamic Programming (DP) approach will be taught in the next class! Today, we focus on the efficiency trap of the Greedy approach.
